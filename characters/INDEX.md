## 使用ルール

1. **主人公の名前は「はな」で統一**（「あや」は使用禁止）
2. **セリフにキャラクターの固有名詞を含めない**
3. **女性陣（はな・さき）**: 黄色の角ばった多角形吹き出し
4. **男性陣（まさと・ともや・YOTA）**: パステルブルーの柔らかい多角形吹き出し

## 参照方法

画像生成時にこれらのファイルをリファレンスとして使用してください：

```
characters/hana.png  → 主人公・はなの表情シート
characters/saki.png  → 友人・さきの表情シート
characters/masto.png → 彼氏・まさとの表情シート
characters/tomoya.png → 友人・ともやの表情シート
characters/yota.png  → 弟・YOTAの表情シート
```
